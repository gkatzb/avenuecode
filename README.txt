** This project was structured on Laravel Framework. **

// Needs composer installed

To run it, please run the follow commands:

1. php artisan migrate
2. php artisan db:seed
3. php artisan route:clear
3. php artisan route:cache

To access it from a web server, htaccess must be configured to point to '/public' directory. Also, please check file access permissions.
